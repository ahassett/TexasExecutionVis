All relevant files:
1. index.html
The homepage to open.
2. main.js
The main javascript that sets up general configurations of visualization
3. overview\js_vis_overview.js
The script that contains the function updating the visualization of page "introductory chronicle."
4. geo\js_geo_dead_dots.js
The script that contains the function updating the visualization of Texas map on page "geographical animation."
5. dashboard\js_vis_dashboard.js
The script that contains the function updating the visualization of brushing Trellis plot.
6. _data\Execution.csv
The original dataset to import
7. geo\Texas_map.html
The html that contains Texas map to import
8. _css\*
Everything in _css folder is stylesheet for different components.

Server instruction
1. Run Git Bash on the repository folder
2. type "python -m http.server"
3. Open a incognito/private page in Chrome/Firefox
4. Go to the url: http://127.0.0.1:8000
5. To shut down the server, go back to Git Bash and hit Ctrl + C or Command + C
